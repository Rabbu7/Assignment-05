Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, allowing programs to interact with, access, and modify the content, structure, and styling of the document dynamically.

How do you select an element from the DOM?

The DOM elements can be selected using methods like getElementById(id) for an element by its id, querySelector(selector) for the first matching CSS selector, and querySelectorAll(selector) for all matches. Other methods include getElementsByClassName(class) and getElementsByTagName(tag) for class and tag-based selections.

What is event delegation in the context of DOM, and why is useful?

Event delegation is a technique where a single event listener on a parent element handles events for its child elements, leveraging event bubbling. It's useful for improving performance and managing dynamic content.

How do you manipulate an element's attributes and styles using DOM?

We can manipulate an element's attributes using methods like setAttribute() and getAttribute() or by directly modifying properties like element.id or element.src. For styles, use element.style.property to change specific CSS properties.
